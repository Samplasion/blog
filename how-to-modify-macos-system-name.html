How to modify the macOS system name and icon

Posted · 2 minute read
Update 17/11/19: This post was updated with info pertaining how to change the icon

Have you ever felt the need to rename macOS to whatever you like? Like, suppose you've just adopted a dog and you're wondering: “Wouldn't it be cool if I called my OS "dogOS"?”. Well, Today, I'm here to make your dreams come true.

Important Notice: I am not responsible for any damage caused to your system. Be cautious when following this tutorial.

The first step will actually be to download the necessary files. We'll need optool, iHex and Theme Engine.

Next, go to the desktop and make two new folders, 'Originals' and 'Modified', then put optool into 'Modified'.

<p>Go to /Applications/Utilities/System Information.app/Contents/MacOS and copy the System Information binary into both of those folders (the “Original” folder will serve as backup, so make sure to NOT ever lose that).</p><p>Open Terminal, then run cd ~/Desktop/Modified and run ./optool strip -w -t System\ Information to strip the code signature (this is needed to run the binary, as macOS refuses to open binaries signed with a wrong certificate).</p><p>Open the System Information file in iHex, use ⌘F and switch to “Text” mode to find the macOS <version name> (replace <version name> with your version’s official name, note that it is case-sensitive). Replace it with what you want, as long as it’s the same length as before (spaces do count, so assume that you’re on Mojave, your total string would need to be 12 in length). Look for just macOS and change the instance of ‘macOS’ that comes up near OS_VERSION_BUILD to match with the first part of the string you changed earlier (I did macOS -> samOS). Don’t forget to save (Depending on the system configuration it may or may not ask for a password).</p><p>Now copy this modified System Information binary to /Applications/Utilities/System Information.app/Contents/MacOS and then enter your password to finalize the process.</p><p>Next, to edit the icon, there are two methods, depending on the version. If one doesn’t work, move on to the next.</p><ol><li>Go to /Applications/Utilities/System Information.app/Contents/Resources and copy the file Assets.car to both “Original” and “Modified”. Then, open ~/Desktop/Modified/Assets.car with Theme Engine and replace any stock icon with the one of your choice. Save and replace the one in /Applications/Utilities/System Information.app/Contents/Resources with it.</li><li>Go to /Applications/Utilities/System Information.app/Contents/Resources and copy every icon you want to modify to “Original”. Then, convert every icon you want to replace to the target format of the stock icon (for example, SystemLogo.tiff is a tiff file, so I’ll want to convert my replacement to tiff) and rename it exactly like the stock one. Then, drag the modified icon into /Applications/Utilities/System Information.app/Contents/Resources and confirm by entering your password.</li></ol><p>Once you’ve done that, the only thing that’s left to do is see if it works! Open the About this Mac menu ( > About this Mac) and watch magic happen under your eyes (you may have to ⌘Q it and retry). If it didn’t work, make sure you followed every step correctly (in every case you can restart the process by restoring the backup). Ok, for this post it’s all, I hope to see you in the next one! See ya!</p>Well, technically, I was helped in writing part of this post by Discord user Alex Fox#0346, so credit goes to him
